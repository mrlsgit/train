## <center>class 2</center>
---
1. **超链接a**:
   + 跳转方式：target 属性
      + _blank 新建页面打开
      + _self 在当前窗口打开
    + 本页面作为锚点,`#+目标id值`：
       ```
       <a href="#case4"> test </a>
       <div id="case4"></div>
       ```
    + title 属性，鼠标悬停时的提示，可以用中文
2. **通用属性**
   + id属性（全页面属性）
      + id的值应当是页面唯一的
      +  不能用中文
   + class属性
      + 类名可以多处使用
3. **路径**
   + 绝对路径
      + 浏览器搜索栏中的路径
   + 相对路径
      + 当前页面文件所在文件夹：`./`
      + 当前页面文件所在文件夹的上一级文件夹： `../`
      + 上级目录的上级目录： `../../`
